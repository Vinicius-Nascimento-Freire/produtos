# produtos
código responsável pela análise de portfólio de produtos

#Meu Primeiro repositório
Entrada Dinâmica: Permite cadastrar múltiplos produtos em uma única execução.

Cálculo Automático: Calcula o lucro total com base na diferença entre custo e venda, multiplicada pela quantidade.

Armazenamento em Memória: Utiliza listas e tuplas para manter os dados organizados durante a execução.

Relatório Formatado: Exibe uma tabela limpa e alinhada ao final da análise.
